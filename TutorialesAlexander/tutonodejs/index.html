Tutorial Instalación y uso de Node JS

En el siguiente tutorial, explicaremos cómo instalar Node JS en una terminal de Cloud9 y un ejemplo de su uso.

Instalación en Cloud9

  1. Para instalar Node JS en Cloud9, en nuestra cuenta de usuario, creamos un nuevo espacio de trabajo.

  2. Poner un nombre, node-js.

  3. Seleccionamos el template de Node JS.

  4. Tendremos un espacio de trabajo con Node JS listo para usar.

Ejemplos de uso

Con Node JS podremos hacer bastantes cosas. En este tutorial explicaremos una sencilla.

  1. En la terminal escribimos: node
  2. Luego escribimos: console.log(‘Hola,Mundo!’)
  3. Y nos mostrará por pantalla: Hola,Mundo! undefined
  4. Podemos crear un archivo llamado holamundo.js con el comando anterior y ejecutarlo. Nos dará el mismo resultado
