Instalación de NodeJS en Windows.

  1. Accedemos a la página web de NodeJS https://nodejs.org/en/. Pinchamos en la versión v.5.6.0 Stable y empezará a descargarse el programa.
  2. Una vez descargado el programa, abrimos el programa de instalación que nos irá guiando a lo largo del proceso.



  3. Se nos instalarán todos los componentes necesarios del programa.

Instalación de GitHub App para Windows.

  1. Accedemos a la página web https://desktop.github.com/ y comenzamos a descargar el programa.
  2. Ejecutamos el programa de instalación y comenzará a instalarse automáticamente.
  3. Por último, abrir la aplicación y estará lista para usar.

Registro en Cloud9

  1. Accedemos a la página https://c9.io/ y pinchamos en “Sign In”.
  2. Si ya tenemos cuenta, rellenamos los datos y accedemos, sino, pinchamos en “Sign Up” y nos damos de alta.

Instalación Atom en Windows

  1. Accedemos a https://atom.io/ y descargamos la aplicación.
  2. Ejecutamos el programa de instalación y comenzará a instalarse.
  3. Una vez que termina, podemos comenzar con el programa.

Instalación Pandoc en Windows

  1. Accdemos a http://pandoc.org/installing.html y pinchamos en “download page” en la sección de Windows.
  2. Nos iremos al un repositorio en GitHub y tendremos que descargar la versión de Windows.
  3. Ejecutamos el programa de instalación y seguimos los pasos.
