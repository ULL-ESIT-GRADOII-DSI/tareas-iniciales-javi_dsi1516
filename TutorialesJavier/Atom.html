Tutorial instalación Atom en Windows.

  1. Para descargarnos el editor Atom nos dirigimos al siguiente link el cual nos llevará a la página web de Atom donde podremos descargar la aplicación.

  2. A continuación en la página de bienvenida podemos elegir si descargar la version de Windows o de otro sistema operativo. Una vez elegida la aplicación y descargada, ejecutamos el instalador.

  3. El instalador automatiza todo el proceso sin pedirnos confirmación por lo que una vez que finalice lanzará la interfaz del editor con lo que damos por finalizada la instalación.
