Tutorial instalación Node.js en Windows.

  1. Para comenzar deberemos descargarnos la aplicacion para Windows de Node.js, para ello nos dirigimos al siguiente link, lo cual nos llevará a la página web oficial donde podremos descargar el instalador asi como consutar material de referencia.

  2. Una vez elegido el instalador y descargado procedemos a ejecutarlo para comenzar con la instalación.

  3. El instalador nos guiará por la instalación pidiendonos que aceptemos la licencia así como los componentes que se instalarán.

  4. Finalmente ya tendremos instalado Node.js.

  5. Para instalar express debemos de abrir el terminal de nuestro sistema operativo y ejecutar el comando npm install express -- save. Con este comando habremos instalado express si todo ha salido bien como se ve en la siguiente imagen.
