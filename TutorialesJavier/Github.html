Tutorial instalación GitHub en Windows.

  1. Para comenzar deberemos descargarnos la aplicacion de escritorio de Gith Hub, para ello nos dirigimos al siguiente link, lo cual nos llevará a la página web de GitHub para escritorio donde podremos descargar la aplicación.

  2. A continuación en la página de bienvenida podemos elegir si descargar la version de Windows o de Mac. Una vez elegida la aplicación y descargada, ejecutamos el instalador.

  3. El instalador nos pedirá confirmación, aceptamos y la instalación continuará sola hasta el final. Una vez finalizada, pasamos a configurar la aplicación.

    1. En primer lugar introduciremos los datos de nuestra cuenta.
    2. En el siguiente paso eligiremos la configuración y por último los repositorios que incluiremos.

  4. Finalmente ya tendremos configurado GitHub para escritorio.
