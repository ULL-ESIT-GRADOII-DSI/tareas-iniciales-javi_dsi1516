Tutorial instalación Pandoc en Windows.

  1. Para comenzar deberemos descargarnos plugin de Pandoc para la consola de windows, para ello nos dirigimos al siguiente link, lo cual nos llevará al repositorio de GitHub donde podremos descargar la aplicación.

  2. Una vez descargada la version que hayamos elegido ejecutamos el instalador.

  3. El instalador nos pedirá confirmación, aceptamos y la instalación continuará sola hasta el final. Una vez finalizada, ya tendremos disponible Pandoc para ejecutar desde la consola de Windows.

  4. Desde el menú de Inicio de windows podemos acceder al fichero de consulta con los comandos a utilizar para convertir los diferentes ficheros markdown.
    Yo personalmente he ustado usando stackedit.io, un editor online de markdown que permite exportar los ficheros en diferentes formatos, asi como importarlos.
